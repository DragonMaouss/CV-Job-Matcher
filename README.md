# CV-Job-Matcher

Projet de matching intelligent de CV et offres d'emploi.

## Objectifs 

Construire une application capable de : 

- extraire les informations d'un CV
- comparer un CV à une ou plusieurs offres d'emploi
- calculer un score de similarité
- identifier les compétences en commun et les compétences manquantes
